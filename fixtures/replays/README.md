# Replay Fixtures

Replay fixtures provide stable examples for Phase 7 structural replay and comparison behavior.

- `structural-success.json`: a metadata-only replay that matches `fixtures/traces/failed-model-call.json`.
- `divergence-comparison.json`: comparison output for a replay with structure, timing, token, destination, and error drift.
- `policy-change-comparison.json`: comparison output for a replay where policy decisions changed.
